#### 🌟 開發亮點
- **成長軌跡**：此網站是我前端進修初期（學習未滿兩個月）的作品。
- **跨越挑戰**：在極短時間內從設計及程式新手，高強度同步吸收新知，獨立完成從 UI 設計到 RWD 網站開發的首項個人專案。

#### 📝 網站簡介
旨在降低學習樂器的入門門檻。針對「想練習但無樂器」或「樂器不便攜帶」的痛點，打造一個能方便、快速借到樂器的平台，讓使用者即時享受音樂帶來的樂趣。

#### 🎨 品牌設計理念 (Logo Design)
我將「自由、動感、專業」三種元素融入品牌識別中：
* **自由與溫度**：大筆觸漸層字母 **Z**，象徵租借規定的高度自由；由藍轉橘的漸層色調，強調年輕、創意與有溫度的服務形象。
* **音樂與即時性**：右側結合**音波/波形**設計，暗示平台的即時性（聲波動感 = 快速租借），直觀傳遞音樂核心。
* **產業辨識度**：融入**鋼琴黑鍵**形狀，強化樂器租借平台的專業第一印象。
